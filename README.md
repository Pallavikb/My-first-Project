# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview
This project analyzes pizza sales data using SQL to generate meaningful business insights. The analysis helps understand customer ordering behavior, revenue trends, and product performance, enabling better business decisions. :contentReference[oaicite:0]{index=0}

## 🎯 Objectives
- Analyze total orders and revenue
- Identify top-selling pizzas
- Find the highest-priced pizza
- Analyze sales by category and size
- Identify peak ordering hours
- Calculate cumulative revenue
- Determine revenue contribution of each pizza type

## 🛠️ Tools Used
- SQL (MySQL)
- MySQL Workbench
- GitHub

## 📂 Dataset
The dataset contains:
- Orders
- Order Details
- Pizzas
- Pizza Types

## 📊 SQL Analysis Performed
- Total number of orders
- Total revenue generated
- Highest-priced pizza
- Most common pizza size
- Top 5 ordered pizza types
- Total quantity sold by category
- Orders by hour
- Category-wise distribution
- Average pizzas ordered per day
- Top 3 pizzas by revenue
- Revenue contribution percentage
- Cumulative revenue over time
- Top 3 pizzas by revenue in each category :contentReference[oaicite:1]{index=1}
